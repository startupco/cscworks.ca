Customization

1) How To Change Favicons ?

Answer: Open your index.html file then go to code no 16 then change the href with your icon or image link as you want.

                        <!-- favicon -->

    <link rel="shortcut icon" type="image/x-icon" href="assets/img/logo/favicon.png">
2) How To Change logo ?

Answer: Open your index.html file then go to code no 47 then change the image source file as you want.

                    <a class="navbar-brand" href="index.html">

                    <img src="assets/img/logo/logo.png" alt="logo">

                </a>
3) How To Change Copy Right ?

Answer: Open your index.html file then go to code no 1306 then change the Copy Right as you want.

                        <p class="copyright-text">

                            &copy; Copyright <span id="date"></span> <a href="#"> COSPACE </a> All Rights Reserved.

                        </p>
4) How To Change Contact Form Email ?

Answer: Open your cospace/assets/php/contact.php file then go to code no 17,18 then change the email as you want.

	 $to = "info@cscworks.ca"; //Replace your real receiving email address

	 $header = "From:info@cscworks.ca "; //Replace with your real web master email
